# 虚拟战士手办柜

> AI人格种子 × 异步生命机制

一个关于玩具苏醒、AI学习与记忆延续的游戏概念展示页面。

## 🎯 项目概念

"当你离开时，他们仍在活着。"

这是一个探索AI人格化和异步生命机制的设计实验，旨在让童年幻想的"玩具会动"成为一种真实的体验。

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm start
```

### 构建生产版本
```bash
npm run build
```

### 部署到GitHub Pages
```bash
npm run deploy
```

## 🛠 技术栈

- **React** - 用户界面框架
- **Framer Motion** - 动画库
- **Tailwind CSS** - 样式框架
- **GitHub Pages** - 部署平台

## 📱 功能特性

- 🎨 响应式设计，支持移动端
- ✨ 流畅的滚动动画效果
- 🌙 动态昼夜转换
- 💫 粒子背景效果
- 🤖 AI梦境模拟器
- 📊 系统流程可视化

## 🎮 核心机制

### AI人格种子系统
每个手办都有独特的人格种子，包含：
- 语言风格 (tone)
- 行为倾向 (core_trait)
- 偏好记忆 (bias_memory)
- 小缺陷 (defect)

### 异步生命机制
手办在玩家离线时仍会：
- 换姿势、放纸条
- 生成梦境或留言
- 自发对话或合照
- 创建纪念场景
